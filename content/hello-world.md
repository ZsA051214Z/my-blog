# Hello World

欢迎来到我的个人博客！这是我写的第一篇文章，用来测试博客系统是否正常运行。

## 为什么开始写博客？

在这个信息爆炸的时代，写博客似乎有些"复古"。但我觉得，写作是最好的思考方式之一。

> 写作不是因为有话要说，而是因为有话要发现。 —— E.M. Forster

## 这个博客用什么搭建？

这个博客完全使用**纯静态技术**搭建：

- `HTML5` - 语义化标签
- `CSS3` - CSS 变量、Grid、Flexbox
- `Vanilla JavaScript` - 零框架依赖
- `Markdown` - 内容管理
- `marked.js` - Markdown 解析
- `highlight.js` - 代码高亮

```javascript
// 一个简单的问候
function greet(name) {
  return `Hello, ${name}! 欢迎来到我的博客 🎉`;
}

console.log(greet('World'));
```
